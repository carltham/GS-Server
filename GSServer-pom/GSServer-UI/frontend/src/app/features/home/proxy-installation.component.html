1. Install NGINX

  1. Update package lists: sudo apt update
  2. Install the server: sudo apt install -y nginx
  3. Enable at boot and start now: sudo systemctl enable nginx, then sudo systemctl start nginx
  4. Confirm it is running: sudo systemctl status nginx

2. Configure the reverse proxy

  1. Create the site file /etc/nginx/sites-available/gsserver-proxy with the server block below. Replace UPSTREAM_HOST and UPSTREAM_PORT with the upstream target you set on the Proxy Management page. 
    server {{ '{' }}
    listen 80;
    server_name _;

    location / {{ '{' }}
        proxy_pass http://UPSTREAM_HOST:UPSTREAM_PORT;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    {{ '}' }}
{{ '}' }}
    Tip: from the terminal you can write it in one step — sudo nano /etc/nginx/sites-available/gsserver-proxy.
  2. Enable the site: sudo ln -sf /etc/nginx/sites-available/gsserver-proxy /etc/nginx/sites-enabled/gsserver-proxy
  3. Validate the configuration: sudo nginx -t
  4. Apply it without downtime: sudo systemctl reload nginx
  5. Return to Proxy Management and re-run detection to confirm the proxy is live.
